
1. Install terraform & check version
    terraform version

2. Create terraform directory and create provider.tf and main.tf
    then run
        terraform init
        terraform plan

3. Terraform Infrastructure Creation order
    VPC
    ↓
    Public Subnet
    ↓
    Internet Gateway
    ↓
    Route Table
    ↓
    Security Group
    ↓
    EC2 Instance
    ↓
    Elastic IP: 15.206.49.189

4. Create SSH key locally Powershell command
    ssh-keygen -t ed25519 -f "$HOME\.ssh\jobseeker-aws"

5. Create ec2 and keypair and security group
    terraform init
    terraform plan
    terraform apply
    Everything will be created

6. SSH into the newly created ec2 Instance
    ssh -i ~/.ssh/jobseeker-aws ubuntu@3.110.182.190

7. Paste this in ansible/inventory.ini
    [aws]
3.110.182.190 ansible_user=ubuntu ansible_ssh_private_key_file=C:/Users/Legion/.ssh/jobseeker-aws

8. Wsl terminal ping
    ansible all -i inventory.ini -m ping

9. Wsl terminal after ping
    ansible-playbook -i inventory.ini playbook.yml

10. SSH into new ELB Instance
    ssh -i ~/.ssh/jobseeker-aws ubuntu@15.206.49.189

11. Generate certificates using certbot
    sudo docker compose run --rm certbot certonly \
  --webroot \
  -w /var/www/certbot \
  -d api.jobseeker.lemongautam.com.np \
  -d grafana.jobseeker.lemongautam.com.np \
  --email lemongautam79@gmail.com \
  --agree-tos \
  --no-eff-email

12. Update api.conf and grafana.conf
    server {
    listen 80;
    server_name api.jobseeker.lemongautam.com.np;

    location /.well-known/acme-challenge/ {
        root /var/www/certbot;
    }

    location / {
        return 301 https://$host$request_uri;
    }
}

server {
    listen 443 ssl;
    http2 on;

    server_name api.jobseeker.lemongautam.com.np;

    ssl_certificate /etc/letsencrypt/live/api.jobseeker.lemongautam.com.np/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/api.jobseeker.lemongautam.com.np/privkey.pem;

    client_max_body_size 50M;

    location / {
        proxy_pass http://backend:7000;

        proxy_http_version 1.1;

        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }
}
